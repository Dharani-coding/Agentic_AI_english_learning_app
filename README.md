# Kili English Learning App

<img src="images/kili_logo.png" alt="Kili Logo" width="100"/>

Learn English Through Conversation, Reflection, and Recall

**Description:**
Powered by agentic AI, Kili (கிளி), named after the parrot in Tamil, is an interactive English learning assistant that helps you improve through natural conversation. Chat freely with Kili, build real-time dialogues, and get smart feedback on grammar, vocabulary, and phrasing. Every conversation gets recorded and analyzed, with insights saved in a local database. Review your progress with random quizzes that test your memory and help you use better language.

**Prerequisite:** 
Please ensure [FFmpeg](https://ffmpeg.org/download.html) is installed and its binary path is added to your system's environment variables (PATH), so the application can access it from the command line.
